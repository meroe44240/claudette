## Format & import Propium

Les 3 CSV sont au format direct-importable dans Propium via `POST /api/v1/import/upload?entityType=candidat` :
- Header : `nom;prenom;email;telephone;poste;entreprise;localisation;linkedin;source;tags;notes`
- Colonnes `nom/prenom/email/telephone/linkedin` vides → à enrichir via FullEnrich avant dialler
- Infos business consolidées dans `notes` (poste/signal, date, URL, effectif, CA, salaire, contexte)
- Tags exploitables côté Propium pour filtrage : vertical, type (JOB/NEWS), type de contact, tranche effectif, ordre contact
